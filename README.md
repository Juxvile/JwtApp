# Rest Jwt Test Application
Это тестовое REST приложение с использованием Spring Security, аутентификация в котором использована с помощью JWT токена. 

####Technologies:
- Java 11
- Maven
- Spring Boot
- Spring Security
- MySQL
- Lombok
- Liquibase

####Endpoints:

#####Post
- /api/v1/auth/login

#####Get
- /api/v1/admin/users/{id}
- /api/v1/users/{id}
